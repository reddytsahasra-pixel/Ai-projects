📄 Overview

This project demonstrates a basic AI agent built using LangChain and Gemini with tool integration and conversational memory.

🧠 Core Idea

The system enhances a language model by adding:

External tool usage (search function) Memory for multi-turn conversations Agent-based decision making ⚙️ Key Features Tool-augmented responses Continuous conversation loop Memory using thread-based session ID 🏗 Flow User → Agent → Decision (Tool / Direct Answer) → Response → Memory Storage 📌 Conclusion

Introduces the foundational concept of AI agents that combine reasoning, tools, and memory using LangChain.
